Installation:
1. Import database using SQL file "database.sql" in phpmyadmin
2. (Optional) populate database with sample data. Run "sample_data.sql" on the database
3. Paste the aussie-car-sales folder onto htdocs/ folder for your XAMPP/WAMP/MAMP installation

Usage:
- Registration can be done using Register link on the navbar
- Login is necessary after registering a new user
- Listing car for sale is not accessible unless you Login
